Section VI

If you're curious about how I made that table in the previous section look so great, I used this HTML:

<table bgcolor="#000">
	<colgroup>
		<col span="2" bgcolor="#ffffff">
	</colgroup>
	<tr>
		<th>Mountain</th>
		<th>Height (ft)</th>
	</tr>
	<tr>
		<td>Mt. Whitney</td><td>14,505</td>
	</tr>
	<tr>
		<td>Mt. Rainier</td><td>14,411</td>
	</tr>
	<tr>
		<td>Mt. Shasta</td><td>14,179</td>
	</tr>
</table>
